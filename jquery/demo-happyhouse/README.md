## 후기

나 임규태, 대장으로써 진두지휘하며 오늘 하루 종일 **'아직 포기하지 않았다'**는 마음으로 참여하고 싸피 지각생을 독려하며 프로젝트를 완수하였다. 

나 이연주는, 쁘띠뀨태 대장님과 함께 화면 디자인도 하고 부트스트랩으로 몇 시간만에 **싸지방** 화면을 뚝딱 만들었다. 저녁밥먹으러 나갔다가 컴퓨터가 꺼져서 뛰어오는 열의를 보여드려서 칭찬받았다. 아주 아주 재밌는 시간이었다.
